My Work

CycleGlasgow

For a college project, I was tasked with developing a dynamic website that promotes cycling in Glasgow. The website offers various functionalities, such as user registration and login, posting cycling-related listings (similar to Gumtree), displaying a cycle path map, and providing access to blog entries. Additionally, the website utilizes CouchCMS, enabling the client to update the blogs easily.
